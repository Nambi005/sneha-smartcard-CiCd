# Java CI/CD Demo Project

This project demonstrates a simple CI/CD pipeline using Java, Maven, JUnit, and GitHub Actions.

## What this project does
- Uses Java classes based on a smart card journey system
- Includes automated tests using JUnit
- Runs tests automatically whenever code is pushed to GitHub

## Why this is useful
This project helped me understand:
- version control with GitHub
- automated testing
- CI/CD workflows
- how code changes can trigger builds and tests automatically
